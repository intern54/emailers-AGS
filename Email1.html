วิธีการใช้งาน

1. เปิดเว็บเบราว์เซอร์และไปที่ URL: https://avt-vms.com

2. ป้อน User ID และ  Password ของท่านในช่องที่กำหนด

3. คลิกที่ปุ่ม "เข้าสู่ระบบ"

4. เมื่อเข้าสู่ระบบแล้ว กรุณาเลือกเมนู "Vendor Profile"

5. กดที่ปุ่ม "Edit" เพื่อทำการอัปเดตข้อมูลที่ต้องการแก้ไขให้ครบถ้วนและถูกต้อง ข้อมูลใดที่มี * เป้นข่อมูลสำคัญต้องอัพเดตไม่อย่างนั้นจะบันทึกไม่ได้

6. ผู้ขายสามารถเปลี่ยน password ได้ในขั้นตอนนี้ โดย password ต้องมีอย่างน้อย 8 ตัวอักษร มีอักขระพิเศษ และ ตัวอักษรพิมพ์ใหญ่ อย่างน้อย1 ตัว

7. เมื่อทำการอัปเดตเสร็จสิ้น กรุณาคลิกที่ปุ่ม "บันทึก" เพื่อบันทึกข้อมูล

8. หากท่านมีข้อสงสัยหรือต้องการความช่วยเหลือเพิ่มเติม กรุณาติดต่อทีมงานของเราได้ที่  avt.supportvms@agc.com
